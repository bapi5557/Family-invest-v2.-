# **App Name**: KinVest

## Core Features:

- Unified Family Dashboard: Real-time overview of all family members and aggregate spending totals powered by Cloud Firestore sync.
- Categorized Expense Tracking: Granular ledger for tracking utilities, groceries, and medical costs with support for instant CRUD operations.
- AI Insight Counselor Tool: A generative AI feature that uses the spending data as a tool to reason through household expenses and provide monthly efficiency summaries.
- Secure Multi-User Auth: Complete authentication flow including registration, login, and forgot password using Firebase Authentication.
- Member Profile Management: Individual profiles for family members featuring contact details and personal investment notes.
- Digital Performance Exports: Generate and share PDF reports of monthly household expenditure for transparent family record-keeping.
- Real-time Persistence: Direct integration with Cloud Firestore to ensure all family data stays in sync across multiple mobile and desktop devices.

## Style Guidelines:

- Primary color: Deep Reliable Navy (#2950A3), chosen for its association with financial stability and trust.
- Background: Clean Slate (#F8F9FB), a heavily desaturated version of the primary navy to maintain professional clarity.
- Accent color: Saturated Teal-Cyan (#27A3C7), providing a vibrant contrast for call-to-action elements.
- Font pairing: 'Alegreya' (serif) for dignified section headlines, and 'Inter' (sans-serif) for high-readability body text and financial digits.
- Fine-line, geometric outline icons used consistently to signify financial categories like utility bills, rent, and education.
- A focused mobile-first approach utilizing spacious card components and bottom navigation for effortless one-handed use.
- Fluid, meaningful micro-interactions for financial calculation updates and smooth card transitions between the dashboard and profile views.
