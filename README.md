
# KinVest - Shared Family Ledger

Secure and efficient family investment and expense management.

### Public Access
This application is deployed to Firebase Hosting and can be accessed publicly at:
**[https://studio-7478833500-c0c46.web.app](https://studio-7478833500-c0c46.web.app)**

### Deployment
To deploy updates to this site, run:
`npm run deploy`

### Features
- **Shared Family Account**: One login for the whole family.
- **Shared Ledger**: Track all household expenses and income.
- **Family Reminders**: Never miss a bill or family event.
- **AI Financial Intelligence**: Get insights into spending patterns.
- **Data Export**: Share reports via PDF or CSV.
